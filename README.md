# 🎮 Tic Tac Toe AI with Minimax Algorithm

[![Live Demo](https://img.shields.io/badge/Demo-Live%20Game-green?style=for-the-badge&logo=vercel)](https://tic-tac-toe-flux30s-projects.vercel.app/)
[![License](https://img.shields.io/badge/License-MIT-blue?style=for-the-badge)](LICENSE)

A responsive, dark-themed Tic Tac Toe game featuring an **unbeatable AI** powered by the Minimax algorithm. Play against perfect AI strategy or challenge friends!

## ✨ Features

- **🤖 Unbeatable AI** - Uses Minimax algorithm to never lose (always wins or draws)
- **🌙 Dark Mode UI** - Sleek, eye-friendly interface
- **⚙️ Customizable Start** - Choose to play first (X) or let AI start (O)
- **🚀 Instant Feedback** - Win/draw detection with smooth animations
- **🔄 Restart Any Time** - Quick game reset option

## 🛠️ Technologies Used

- **Frontend**: HTML5, CSS3, JavaScript (ES6)
- **Algorithm**: Minimax with recursion
- **Hosting**: Vercel

## 🎯 How to Play

1. **Choose who starts** (You or AI)
2. **Click any empty cell** to place your mark (X)
3. **Watch the AI respond** with optimal moves (O)
4. **Win** by getting 3 in a row, or **force a draw**!
